# Showcase to develop a web-app with frontend and backend

Development of a simple Kanban-style to-do-list.
Deployment on AWS Amplify Hosting with the help of the git-repo.
Use Node.js.

The spec should contain a todo-list that is structured in a way that after each tasks there's something to deploy and to test. Each task has a status "open" or "closed" and detailed acceptance criteria.
